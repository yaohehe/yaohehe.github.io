## Step 1: Domain Registration and Hosting

### Choosing the Right Domain Name

Your domain is your website address (e.g., example.com). Tips for picking one:

- Keep it short: Long domains are hard to remember

  • Include keywords: If you're in a specific niche, relevant words help
  • Avoid hyphens: my-site.com looks unprofessional
  • Stick to .com: It's the most trusted TLD

## Step 2: Installing WordPress

### One-Click Install (Shared/Managed Hosting)

Most hosts offer instant WordPress installation:

1. Log in to your hosting control panel (cPanel or custom) 2. Find "Softaculous" or "WordPress Installer" 3. Click Install, fill in site name and admin account 4. Wait 2-5 minutes—done!

### Manual Installation (VPS/Cloud Server)

For cloud servers, you'll need to install manually:

`bash

SSH into your server

ssh root@your_server_ip

# Install LAMP stack apt update && apt upgrade -y apt install apache2 mysql-server php php-mysql -y

# Download WordPress cd /var/www/html wget https://wordpress.org/latest.tar.gz tar -xzf latest.tar.gz rm latest.tar.gz

# Set up database mysql -u root -p CREATE DATABASE wordpress; CREATE USER 'wpuser'@'localhost' IDENTIFIED BY 'your_password'; GRANT ALL PRIVILEGES ON wordpress.* TO 'wpuser'@'localhost'; FLUSH PRIVILEGES; EXIT; `

## Step 3: Setting Up SSL Certificate (HTTPS)

Google uses HTTPS as a ranking signal, and browsers now flag non-HTTPS sites as "not secure."

Free Option: Let's Encrypt `bash apt install certbot python3-certbot-apache -y certbot --apache -d yourdomain.com -d www.yourdomain.com `

Auto-renewal: `bash systemctl status certbot.timer # Verify timer is active `

## Step 6: Basic SEO Setup

### 1. Submit Your Sitemap After installing Yoast SEO, verify your sitemap at yourdomain.com/wp-sitemap.xml, then submit to Google Search Console.

### 2. Set Permalinks Go to Settings → Permalinks, choose "Post name": ` https://yourdomain.com/article-slug/ `

### 3. Configure robots.txt Create robots.txt in your site root: ` User-agent: * Allow: / Disallow: /wp-admin/ Disallow: /wp-includes/ Sitemap: https://yourdomain.com/wp-sitemap.xml `

## FAQ

Q: What's the difference between WordPress.com and WordPress.org?

WordPress.com is a hosted platform (like Wix/Squarespace)—free but limited. WordPress.org is the open-source software you install on your own hosting—full control. Always choose WordPress.org.

Q: What if my site gets hacked?

1. Scan with Wordfence for malicious code 2. Restore from a clean backup 3. Change all passwords (WordPress, database, FTP) 4. Contact your host to verify server security

Q: How can I speed up WordPress?

- Use a caching plugin (WP Super Cache or WP Rocket)

  • Convert images to WebP and compress (ShortPixel or Imagify)
  • Use a CDN (Cloudflare free tier is sufficient)
  • Remove unnecessary plugins
  • Choose a fast hosting provider
