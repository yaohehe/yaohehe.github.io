2 注册域名

域名是你网站的"门牌号"。选择域名时注意：

  • 尽量用.com后缀（最具信任感）
  • 简短易记，不超过15个字符
  • 避免连字符和数字
  • 如果.com被注册，可以考虑.io、.co等替代

第二步：服务器环境搭建

3 选择安装方式

两种主流安装方式：

方式A：一键安装（推荐新手）

大多数云服务商都提供一键安装WordPress的功能：

  1. 登录云服务器控制面板
  2. 选择"Applications"或"Marketplace"
  3. 找到WordPress镜像
  4. 点击创建，等待3-5分钟完成部署

方式B：手动安装（适合有技术基础的用户）

# 1. 更新系统
sudo apt update && sudo apt upgrade -y

# 2. 安装LEMP栈
sudo apt install nginx mysql-server php-fpm php-mysql -y

# 3. 下载WordPress
cd /var/www/html
sudo wget https://wordpress.org/latest.tar.gz
sudo tar -xzf latest.tar.gz

# 4. 配置Nginx和MySQL
# （详细配置请参考官方文档）
4 配置SSL证书

HTTPS是必须的。使用Let's Encrypt免费获取SSL证书：

sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com
✅ 提示：Let's Encrypt证书会自动续期，设置好之后无需再操心SSL问题。

第三步：WordPress初始配置

5 完成安装向导

浏览器访问你的域名，会看到WordPress安装界面：

  1. 选择语言（中文/English）
  2. 填写站点名称、管理员用户名和密码
  3. 填写管理员邮箱
  4. 点击"安装WordPress"
6 选择并安装主题

主题决定了网站的外观。2026年推荐的免费主题：

  • Astra：轻量、快速、高度可定制
  • GeneratePress：性能优化，适合SEO
  • Kadence：内置模板库，新手友好
  • Blocksy：原生Gutenberg支持
7 安装必备插件

以下为每个WordPress站点都应该安装的插件：

  • Yoast SEO 或 Rank Math — SEO优化
  • Wordfence Security — 安全防护
  • WP Super Cache 或 W3 Total Cache — 缓存加速
  • UpdraftPlus — 自动备份
  • Wordfence Login Security — 登录保护

第四步：内容创建与优化

8 创建关键页面

一个完整的WordPress站点需要以下页面：

  • 首页：展示你的品牌和价值主张
  • 关于页面：介绍你和你的团队
  • 联系页面：提供联系方式或表单
  • 隐私政策：法律合规要求
  • 博客：定期更新内容，吸引流量
9 SEO基础优化

发布前做好SEO，让你的内容更容易被搜索引擎收录：

  • 设置固定链接为"文章名"格式（/%postname%/）
  • 每个页面设置唯一的标题和meta描述
  • 使用关键词研究工具（如Google Keyword Planner）找到合适的关键词
  • 确保所有图片都有alt标签
  • 提交sitemap到Google Search Console
10 性能优化

网站速度直接影响SEO排名和用户体验：

  • 启用页面缓存（使用WP Super Cache等）
  • 压缩图片（使用ShortPixel或TinyPNG）
  • 使用CDN加速（Cloudflare免费版即可）
  • 减少不必要的插件
  • 启用GZIP压缩
