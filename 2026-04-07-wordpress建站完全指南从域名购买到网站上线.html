## 第一步：域名注册与主机选购

### 选择合适的域名

域名就是你的网站地址，如 example.com。选择域名时要注意：

- 简短好记：避免过长的域名，用户很难记住

  • 包含关键词：如果做特定行业，最好包含相关词汇
  • 避免连字符：如 my-site.com 尽量不要用连字符
  • 选择主流后缀：.com 是最安全的选择，其次是 .cn、.org

主流域名注册商对比：

### 选择网站主机

主机（Web Hosting）就是存放你网站文件和网络内容的服务器。常见主机类型：

共享主机：最便宜，多个网站共用一台服务器，适合新手入门 VPS 主机：虚拟专用服务器，性能更稳定，适合成长型网站 云服务器：如 AWS、DigitalOcean、Vultr，弹性扩展，适合技术用户 WordPress 专用主机：针对 WordPress 优化，配置简单

对于 TechPassive 推荐的技术类博客，建议选择 DigitalOcean 或 Vultr 的 VPS，价格从 $4-6/月起，性能稳定，全球数据中心可选。

## 第二步：安装WordPress

### 使用主机商的一键安装

大多数主流主机（如 SiteGround、Bluehost、WP Engine）都提供一键安装 WordPress，无需任何技术背景：

1. 登录主机控制面板（cPanel 或自定义面板） 2. 找到 "Softaculous Apps Installer" 或 "WordPress" 3. 点击安装，填写网站名称和管理员账号 4. 点击安装，等待 2-5 分钟完成

### 手动安装WordPress（VPS 用户）

如果你使用的是云服务器，需要手动安装：

`bash

SSH 连接到服务器

ssh root@your_server_ip

# 安装 LAMP 环境（Linux + Apache + MySQL + PHP） apt update && apt upgrade -y apt install apache2 mysql-server php php-mysql -y

# 下载 WordPress cd /var/www/html wget https://wordpress.org/latest.tar.gz tar -xzf latest.tar.gz rm latest.tar.gz

# 配置数据库 mysql -u root -p CREATE DATABASE wordpress; CREATE USER 'wpuser'@'localhost' IDENTIFIED BY 'your_password'; GRANT ALL PRIVILEGES ON wordpress.* TO 'wpuser'@'localhost'; FLUSH PRIVILEGES; EXIT; `

## 第三步：配置SSL证书（HTTPS）

Google 已明确表示 HTTPS 是排名因素之一，未安装 SSL 的网站会被浏览器标记为"不安全"。

免费方案：Let's Encrypt `bash apt install certbot python3-certbot-apache -y certbot --apache -d yourdomain.com -d www.yourdomain.com `

自动续期： `bash systemctl status certbot.timer # 确保定时任务已启用 `

## 第四步：选择合适的主题

WordPress 主题决定了你网站的外观。选择主题时要注意：

- 响应式设计：必须在手机端显示正常（Google 移动优先索引）

  • 加载速度：选择轻量级主题，测试工具用 PageSpeed Insights
  • 定期更新：选择有活跃开发团队的主题
  • 评价与支持：查看用户评价，了解开发者响应速度

TechPassive 推荐免费主题：

  • GeneratePress：轻量、速度快、社区活跃
  • Astra：高度可定制，加载时间 < 0.5s
  • Hello Elementor：配合 Elementor 页面编辑器使用

## 第六步：网站SEO基础设置

### 1. 提交网站地图 安装 Yoast SEO 后，访问 yourdomain.com/wp-sitemap.xml 确认地图已生成，然后提交到 Google Search Console。

### 2. 设置固定链接结构 后台 → 设置 → 固定链接，选择"文章名"结构： ` https://yourdomain.com/article-slug/ `

### 3. 配置 robots.txt 在网站根目录创建 robots.txt： ` User-agent: * Allow: / Disallow: /wp-admin/ Disallow: /wp-includes/ Sitemap: https://yourdomain.com/wp-sitemap.xml `

## 常见问题解答

Q: WordPress.com 和 WordPress.org 有什么区别？

WordPress.com 是托管平台（类似 Wix/Squarespace），免费可用但功能受限。WordPress.org 是开源程序，需要自己购买主机，但完全掌控网站。建议选择 WordPress.org。

Q: 网站被黑客攻击了怎么办？

1. 用 Wordfence 扫描恶意代码 2. 恢复最近的无感染备份 3. 更改所有密码（WordPress、数据库、FTP） 4. 联系主机商确认服务器安全

Q: 如何加快 WordPress 网站速度？

- 使用缓存插件（WP Super Cache 或 WP Rocket）

  • 图片使用 WebP 格式并压缩（ShortPixel 或 Imagify）
  • 使用 CDN（Cloudflare 免费版足够）
  • 减少不必要的插件
  • 选择速度快的主机
