第一步：域名选购——你的网上门牌号

什么是域名，为什么它重要？

域名就是你的网站地址，比如 。它相当于你在互联网上的"门牌号"，用户在浏览器里输入这个地址就能访问你的博客。选择一个好的域名，能让读者更容易记住你，也有利于搜索引擎排名。

第三步：LNMP环境搭建——手把手配置

安装Nginx

SSH连接到你的服务器后，执行以下命令安装Nginx：

sudo apt update && sudo apt install nginx -y
sudo systemctl enable nginx
sudo systemctl start nginx

安装完成后，在浏览器输入服务器IP，应该能看到 Nginx 的默认欢迎页面。

安装MySQL 8.0 或 MariaDB 11

WordPress需要数据库存储内容。2026年推荐使用 （MySQL的社区分支，完全兼容，性能更好）：

sudo apt install mariadb-server -y
sudo systemctl enable mariadb
sudo systemctl start mariadb

# 运行安全配置向导
sudo mysql_secure_installation

配置数据库：

sudo mysql -u root -p

在MySQL命令行中执行：

CREATE DATABASE wordpress_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'wp_user'@'localhost' IDENTIFIED BY '你的强密码';
GRANT ALL PRIVILEGES ON wordpress_db.* TO 'wp_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;

安装PHP 8.2 或 8.3

2026年，PHP 8.2 和 8.3 已经是主流版本。WordPress 6.4+ 对 PHP 8.2 有良好支持，强烈推荐使用 PHP 8.2 或 8.3 以获得更好的性能和安全性：

sudo apt install php8.3-fpm php8.3-mysql php8.3-curl php8.3-gd php8.3-mbstring php8.3-xml php8.3-xmlrpc php8.3-zip php8.3-opcache php8.3-bcmath -y

验证PHP安装：

php -v

应该显示类似：

配置PHP-FPM和Nginx

编辑 Nginx 默认站点配置：

sudo nano /etc/nginx/sites-available/default

关键配置如下：

server {
    listen 80;
    root /var/www/html;
    index index.php index.html;
    server_name your_domain.com;

    client_max_body_size 256M;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/var/run/php/php8.3-fpm.sock;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

    location ~ /\.ht {
        deny all;
    }
}

测试并重载Nginx：

sudo nginx -t
sudo systemctl reload nginx

第四步：安装WordPress——核心操作

下载并配置WordPress

cd /var/www/html
sudo rm -rf *
sudo wget https://wordpress.org/latest.tar.gz
sudo tar -xzf latest.tar.gz
sudo mv wordpress/* .
sudo rm -rf wordpress latest.tar.gz

配置wp-config.php

sudo mv wp-config-sample.php wp-config.php
sudo nano wp-config.php

修改以下数据库配置：

define('DB_NAME', 'wordpress_db');
define('DB_USER', 'wp_user');
define('DB_PASSWORD', '你的强密码');
define('DB_HOST', 'localhost');
define('DB_CHARSET', 'utf8mb4');
define('DB_COLLATE', '');

添加安全密钥

访问 https://api.wordpress.org/secret-key/1.1/salt/ 获取随机密钥，替换wp-config.php中对应的行。

设置文件权限

sudo chown -R www-data:www-data /var/www/html
sudo find /var/www/html -type d -exec chmod 755 {} \;
sudo find /var/www/html -type f -exec chmod 644 {} \;

通过Web界面完成安装

在浏览器访问 ，按照向导填写：

  • 网站标题
  • 用户名（建议不要用 admin）
  • 强密码
  • 邮箱

点击"安装WordPress"，完成！

第五步：SSL证书配置——让网站更安全

2026年，Google等搜索引擎已经把HTTPS作为排名因素，未安装SSL证书的网站会被标记为"不安全"。

使用Let's Encrypt免费证书

sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d your_domain.com -d www.your_domain.com

按照提示输入邮箱、同意条款即可。Certbot会自动配置Nginx并设置自动续期（Let's Encrypt证书有效期90天，Certbot会自动续期）。

安装成功后，访问 ，浏览器地址栏会显示绿色锁标志。
